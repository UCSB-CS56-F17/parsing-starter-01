# parsing-starter-01

This is the starter code for a lab on recursive descent parsing.

See https://ucsb-cs56-f17.github.io/lab/lab08/ for more explanation.

This code uses maven rather than ant.

| To do this ... | type this...  |
|----------------|---------------|
| compile        | `mvn compile` |
| clean up       | `mvn clean`   |
| test           | `mvn test`    |
